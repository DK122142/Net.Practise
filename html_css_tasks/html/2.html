Что такое CMS

CMS - "система управления контентом"(движок) - написанная PHP-программистом основа для сайта, с помошью которой вы можете управлять сайтом(добавлять контент, менять пункты меню и т.п.) не зная HTML и CSS.

Однако, для того чтобы сделать сайт с помошью CMS потребуются услуги и программиста, и дизайнера, и верстальщика. И капиталовложения

Какие бывают cms

Бывают различные системы управления контентом: для интернет-магазинов, для блогов, для форумов и т.д.

Примеры cms

Примеры популярных CMS: Joombla, WordPress(для блогов), PhpBB(для форумов)

CMS-ко бывают платные и бесплатные
